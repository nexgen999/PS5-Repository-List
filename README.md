## 📖 À propos du projet

La scène **PlayStation 5 Underground** évolue extrêmement vite. Ce dépôt a été conçu pour centraliser et indexer de manière chirurgicale tous les acteurs, outils et tutoriels majeurs de la scène. 

En plus des listes de référence, le projet maintient un fichier **OPML universel** permettant de surveiller en temps réel via flux RSS l'activité et les nouveautés de l'ensemble des dépôts.

---

## 🗂️ Index Général des Listes Principales

* 💻 **[dev_list-repository.md](./dev_list-repository.md)** : Regroupe les fiches, profils et dépôts de référence de tous les développeurs majeurs de la scène PS5.
* 📺 **[Community.Tutorial.md](./Community.Tutorial.md)** : Indexation exclusive des **Youtubers**, créateurs de contenu et techniciens de la communauté. Fichier listant l'intégralité des dépôts de ce projet avec la syntaxe exacte de leurs flux RSS/Atom. Pratique pour auditer ou piocher un flux en particulier.
* 📄 **[Consulter le fichier : PS5-Repository-List-with-RSS.md](./PS5-Repository-List-with-RSS.md)**

### 📥 Utilisation avancée du fichier `PS5_Repositories_Feeds.opml`
Le fichier `PS5_Repositories_Feeds.opml` est un standard universel XML (OPML) qui regroupe **tous les flux RSS de la scène PS5** référencés ici. Au lieu d'ajouter manuellement les dizaines de dépôts un par un, ce fichier vous permet de tout configurer en un seul clic !

* 💾 **[Télécharger le fichier prêt à l'emploi : PS5_Repositories_Feeds.opml](./PS5_Repositories_Feeds.opml)**  
  *(Astuce : Clic droit sur le lien -> "Enregistrer le lien sous..." pour le télécharger directement).*

## 🛠️ Guide complet d'utilisation des Flux RSS

Pour lire ce fichier et suivre les mises à jour, vous devez obligatoirement utiliser une **application de lecture de flux RSS** (un agrégateur). 

🚀 Étapes d'installation :
Téléchargez un lecteur RSS :

🖥️ PC / Mac / Linux : Fluent Reader, Akregator, ou utilisez simplement le navigateur Opera qui intègre nativement un excellent lecteur de flux.

📱 Android / iOS : Feeder, ReadYou (Open-source et sans pub) ou NetNewsWire.

☁️ Services Web : Inoreader, Feedly, FreshRSS.

Importez le fichier :

Téléchargez le fichier PS5_Repositories_Feeds.opml depuis ce dépôt.

Ouvrez votre lecteur RSS et allez dans les Paramètres / Options.

Cherchez l'option "Importer" (ou Import OPML / XML).

Sélectionnez le fichier PS5_Repositories_Feeds.opml.

C'est prêt ! Votre lecteur va se synchroniser tout seul et vous recevrez une alerte en direct dès qu'un développeur publiera une nouvelle version (Release) ou un correctif de code, sans avoir à rafraîchir manuellement des dizaines de pages web.

Ce dépôt est un projet vivant maintenu pour la communauté. Si vous constatez un lien mort, un flux RSS expiré, ou si vous souhaitez ajouter un nouveau dépôt, un Youtuber ou un compte X manquant :

Via les Issues : Ouvrez une Issue pour signaler le problème ou suggérer un ajout.

Via Pull Request (PR) :

Forkez le dépôt.

Modifiez directement le fichier concerné (par exemple category/ps5_payloads.md ou Community.Tutorial.md).

Soumettez votre Pull Request pour qu'elle soit validée et fusionnée.
